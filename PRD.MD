# Product Requirements (UTD Hub)

## Goal
Create an English-only resource hub for the Roblox experience **Universal Tower Defense (UTD)**, featuring practical tools (calculators/builders) and evergreen reference pages (codes, tier lists, guides).

## Primary Audience
- New players looking for quick onboarding (codes, basic guide)
- Mid-game players optimizing teams and upgrades (tier list, team builder, upgrade calculators)
- Community members who want safe, curated links (Discord, community hub)

## Content Rules
- English-only content.
- Do not reference or resurrect any previous-site content.
- Do not include exploit scripts, cheating instructions, or bypass methods.

## Key Pages
- Home hub: highlights tools + “how to get started” path
- Codes: active + expired archive, redemption steps, “last checked” timestamp
- Tier List: tiering by role (DPS / Support / Crowd Control / Hybrid) with methodology
- Calculator: DPS / damage planning; upgrade cost planning
- Guide: beginner to advanced progression
- Wiki: systems overview (summons, currencies, modes, traits)
- Community: safe link directory + update checklist
- Discord: safety guidance for finding the official server

## UX Requirements
- Mobile-first layout; accessible contrast and focus states
- Fast, mostly static rendering (SSG-friendly)
- Image-rich sections (hero visuals + diagrams + cards)

## SEO Requirements
- Clear page intent; scannable headings
- Strong internal linking between tools and guides
- Sitemap includes all primary routes

## Non-Affiliation Disclaimer
UTD is a Roblox experience. This site is an unofficial fan resource and is not affiliated with Roblox Corporation or the game’s developers.
